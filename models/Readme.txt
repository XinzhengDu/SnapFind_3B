This model has better support for English. For higher search accuracy, translate your query into English using a translation tool before searching.
Inaccurate results may be due to insufficient frame capture density. You can set the frame interval to 1 second in the File > Settings option.
Inaccurate results may also be caused by using the lightweight model. Larger models deliver better performance yet require more computing power. See: https://github.com/XinzhengDu/SnapFind_3B
